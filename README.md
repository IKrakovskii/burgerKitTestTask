# Условие задачи
```
Предлагаем вам выполнить следующее тестовое задание, потратить на него 1 час своего времени.

Сколько успеете за 1 час, в каком угодно виде (могут быть не прописаны все функции и алгоритмы).

Бот напоминаниий (python-telegram-bot)

Менеджер хочет поставить напоминание сотруднику. Напрмер, заполняет строки в GoogleSheet, указывает

tel_id, тест, дату, время и время на ответ (answer_time).

Сотруднику приходит сообщение с прикрепленной инлайн клавиатурой (кнопка: выполнено / не сделано)

Если сотрудник в течение answer_time нажимает на 1 из кнопок, то менеджеру приходит сообщение и какая кнопка нажата

Если сотрудник в течение answer_time не нажал кнопку - приходит сообщение, что сотрудник проигнорировал.

Всё.

Укажите сколько времени заняло выполнение.
```
